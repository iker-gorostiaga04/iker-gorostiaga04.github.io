# Proyectos

## Reto 1: Integración de Datos desde una API y Análisis con Big Data

**Descripción:**  
Proyecto de integración de datos a partir de una API externa, con el objetivo de crear una base de datos relacional, realizar consultas SQL, generar visualizaciones y documentar todo el proceso de análisis.

**Tareas principales:**

- Conexión a API y creación de bases de datos.
- Análisis de estructura y tipos de datos.
- Diseño de preguntas de negocio para optimizar procesos.
- Consultas SQL (SUM, AVG, COUNT, JOIN, etc.).
- Creación de vistas SQL para facilitar la consulta.
- Análisis de resultados para toma de decisiones.
- Documentación técnica del proceso y resultados.

**Tecnologías:**  
Python, SQL, Pandas, Seaborn, Node-RED, Git

## Reto 2: Monitorización de Datos en Tiempo Real

**Descripción:**  
Desarrollo de un sistema de monitorización en tiempo real utilizando Node-RED para la recopilación y envío de datos a InfluxDB, y visualización de métricas clave a través de dashboards interactivos en Grafana.

**Tecnologías:**  
Node-RED, InfluxDB, Grafana, Flux (2.0)
